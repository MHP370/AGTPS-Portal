# Iran Province Boundaries

`iran-provinces.json` contains simplified ADM1 boundary data for Iran from geoBoundaries.

Source: https://www.geoboundaries.org/

Dataset API used:
https://www.geoboundaries.org/api/current/gbOpen/IRN/ADM1/

The API metadata for this boundary file lists OpenStreetMap/Wambacher as the boundary source and Open Data Commons Open Database License 1.0 as the license.
