# AGTPS Portal

Enterprise Admin Portal built with:

## Stack
- Next.js 14 (App Router)
- NestJS API
- React Query
- TailwindCSS
- TypeScript

## Modules
- Applications CRUD
- Categories CRUD
- Roles & Permissions
- Authentication (JWT)

## Architecture
- apps/web → Frontend
- apps/api → Backend

## API Base
http://localhost:3002/api

## Notes
- Frontend uses React Query for all API calls
- Backend is NestJS with Prisma ORM
- Auth uses JWT Bearer Token
